# RoomWale 🏠

RoomWale is a responsive room rental website built using HTML, CSS, and JavaScript.

## Features

* Responsive Design
* Room Search Functionality
* Room Counter
* Dynamic Room Details Page
* Image Gallery
* About Section
* Contact Form Validation
* Smooth Scroll Navigation
* Footer Section

## Technologies Used

* HTML5
* CSS3
* JavaScript
